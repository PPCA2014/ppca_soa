Serviços que o PPCA API Management traz para você

Clique no nome do serviço para maiores informações:
/user
Corresponde ao serviço que permite listar todos os usuários cadastrados ou inserir um novo usuário
/user/:id
Corresponde ao serviço que permite atualizar um usuário existente, excluir um usuário a partir do id ou retornar informações do usuário a partir do seu id
/catalogo
Serviço que retorna a descrição dos serviços disponíveis no PPCA API Management
/autentica
Serviço que permite a autenticação de um usuário para uso do barramento
/favicon.ico
Serviço que exibe o ícone favicon do Portal PPCA API Management Plataform
/info
Serviço que traz informações sobre o barramento
options
Serviço que retorna os verbos suportados pelo ESB
/samples/pessoa
Serviço que traz informações sobre as pessoas cadastradas no PPCA API Management Plataform.
/samples/pessoa/:id
Serviço que traz informações sobre uma pessoa específica cadastrada no PPCA API Management Plataform a partir do fornecimento do seu id
/samples/contador
Um contador de quantidade de acessos aos serviços do PPCA API Management Plataform
/samples/calc
Calculadora fornecida pelo PPCA API Management Plataform
/samples/hello_world_java
Fornece um "Hello World Java!!!"
/samples/hello_world
Fornece "Hello World" em Erlang
/portal
Corresponde ao acesso ao portal PPCA API Management Plataform
Não encontrado
Corresponde à tentativa de acessar serviços não implementados pelo PPCA API Management Plataform ou serviços inoperantes.
/health/top_services/:top
Corresponde ao serviço de verificação dos serviços mais acessados
/health/qtd_requests_by_date/:top
Corresponde ao serviço de listagem da quantidade de requisições feitas por data.
/health/top_services_by_type/:top
Corresponde à Listagem dos serviços mais solicitados por tipo de verbo HTTP
/
Retorna a mensagem "It works!!!" Caso o barramento esteja operante
